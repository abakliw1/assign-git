Q1. What is a gitignore file and what should it include for a Java project? A. Gitignore files are those files which get ignored while committing any changes on git. The files added in gitignore files are .jar, .class, etc.Q2. What does fast-forward in git mean?A. Fast-Forward in git means merging 2 main/master branches.Q3. Why should you have different branches?A. The use of different branches helps us to do the work parallelly, and if any error persists then the main branch would not get affected because of that.Q4. In a project how many remote Git repositories do you usually have? A. We can add as many numbers of remote repositories as we want.
